# Keyboard Shortcuts

FluxIDE provides VS Code-like keyboard shortcuts for efficient development.

## Global Shortcuts

### File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Save current file |
| `Ctrl+Shift+O` | Open folder |
| `Ctrl+W` | Close current tab |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |

### Panels

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Toggle sidebar |
| `Ctrl+\`` | Toggle terminal |

## Editor Shortcuts

These shortcuts work within the Monaco editor:

### Basic Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+X` | Cut line (empty selection) |
| `Ctrl+C` | Copy line (empty selection) |
| `Ctrl+V` | Paste |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+/` | Toggle line comment |
| `Ctrl+Shift+A` | Toggle block comment |

### Selection

| Shortcut | Action |
|----------|--------|
| `Ctrl+A` | Select all |
| `Ctrl+L` | Select current line |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+Shift+L` | Select all occurrences |
| `Shift+Alt+↑/↓` | Copy line up/down |
| `Alt+↑/↓` | Move line up/down |

### Multiple Cursors

| Shortcut | Action |
|----------|--------|
| `Alt+Click` | Add cursor |
| `Ctrl+Alt+↑` | Add cursor above |
| `Ctrl+Alt+↓` | Add cursor below |
| `Ctrl+U` | Undo last cursor operation |

### Find and Replace

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Find |
| `Ctrl+H` | Find and replace |
| `F3` | Find next |
| `Shift+F3` | Find previous |
| `Ctrl+Shift+F` | Find in files (coming soon) |

### Code Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+G` | Go to line |
| `Ctrl+P` | Quick open file (coming soon) |
| `Ctrl+Shift+P` | Command palette (coming soon) |
| `F12` | Go to definition (coming soon) |

### Code Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+Space` | Trigger suggestions |
| `Ctrl+Shift+Space` | Trigger parameter hints |
| `Tab` | Indent |
| `Shift+Tab` | Outdent |
| `Ctrl+]` | Indent line |
| `Ctrl+[` | Outdent line |

### Folding

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+[` | Fold region |
| `Ctrl+Shift+]` | Unfold region |
| `Ctrl+K Ctrl+0` | Fold all |
| `Ctrl+K Ctrl+J` | Unfold all |

## Terminal Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Interrupt/cancel |
| `Ctrl+D` | End of input (EOF) |
| `Ctrl+L` | Clear terminal |
| `Ctrl+Shift+C` | Copy selection |
| `Ctrl+Shift+V` | Paste |

## macOS Equivalents

On macOS, replace `Ctrl` with `Cmd`:

| Windows/Linux | macOS |
|---------------|-------|
| `Ctrl+S` | `Cmd+S` |
| `Ctrl+Shift+O` | `Cmd+Shift+O` |
| `Ctrl+B` | `Cmd+B` |
| `Ctrl+\`` | `Cmd+\`` |

## Customization

Custom keyboard shortcuts are not yet supported but are planned for a future release.

## Tips

1. **Learn the essentials first**: `Ctrl+S`, `Ctrl+W`, `Ctrl+\``
2. **Use tab navigation**: `Ctrl+Tab` is faster than clicking tabs
3. **Toggle sidebar**: `Ctrl+B` gives you more editor space
4. **Quick terminal access**: `Ctrl+\`` for command-line tasks
