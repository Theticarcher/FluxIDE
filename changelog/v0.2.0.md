# FluxIDE v0.2.0

**Release Date:** 2025-01-23

## Added

- Initial public release of FluxIDE
- Monaco-based code editor with Flux syntax highlighting
- Integrated terminal with PTY support
- Live preview panel for Flux applications
- File explorer with project navigation
- Multi-platform support (Windows, macOS Intel/Apple Silicon, Linux)

## Technical

- Built with Tauri v2 for native performance
- React frontend with TypeScript
- Integrated flux-compiler for real-time compilation
