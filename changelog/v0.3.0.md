# FluxIDE v0.3.0

**Release Date:** 2025-01-26

## Added

### DevTools Panel
Comprehensive debugging tools with 10 specialized tabs:
- **Elements** - DOM inspection and manipulation
- **Console** - JavaScript console output and execution
- **Sources** - View compiled HTML, CSS, and JavaScript
- **Network** - Monitor network requests
- **Performance** - Performance profiling tools
- **Memory** - Memory usage analysis
- **Application** - Local storage, session storage, cookies inspection
- **Security** - Security analysis tools
- **FluxScore** - Flux-specific code quality metrics
- **Recorder** - Record and replay user interactions

### Visual Editor
- Foundation for drag-and-drop Flux UI composition
- Visual editor store for state management
- Component definitions and types

### Panels
- **Problems Panel** - Display compiler errors with line/column information
- **Output Panel** - Show compiler stdout/stderr output
- **Elements Panel** - DOM tree inspection

## Changed

- Integrated DevTools tabs into bottom panel alongside Terminal
- Panel component now supports 13 tabs with icons
- Enhanced Monaco Flux tokenizer with additional keywords and syntax support
- New application icons (32x32, 128x128, 128x128@2x, ico)
- Scrollable panel tabs for better navigation
- Improved panel header layout and styling

## Fixed

### Terminal
- **Character duplication** - Fixed issue where characters were duplicated when typing
- **Backspace behavior** - Backspace now works correctly like native terminals
- **Enter key** - Fixed last character being duplicated when pressing Enter
- **Persistence** - Terminal now persists across panel tab switches (no more disconnections)

### Technical Fixes
- Removed React Strict Mode to prevent double mount/unmount cycles
- Added proper event listener cleanup with race condition protection
- Implemented CSS-based visibility toggle for terminal persistence
- Fixed async Tauri event listener lifecycle management
- Added cleanup state tracking to prevent stale event processing

## Technical

- Added visual-editor-store for Visual Editor state management
- Added visual-editor types for component definitions
- Added flux-components data catalog
- Updated flux-compiler submodule with enhanced error reporting and CI workflow
