# FluxIDE v0.4.0

**Release Date:** 2025-01-26

## Added

### Theme System
Complete theming infrastructure with 6 premade themes:
- **Flux Dark** (default) - Electric violet and cyan accents on deep purple-gray backgrounds
- **Flux Light** - Clean white backgrounds with the same vibrant accent colors
- **Neon Nights** - Cyberpunk-inspired with hot pink and electric blue on black
- **Ocean Breeze** - Deep sea blue backgrounds with teal and cyan accents
- **Forest** - Dark green-brown palette with emerald and gold accents
- **Sunset** - Purple and red backgrounds with orange and magenta gradients

### Settings Modal
New settings UI accessible from the Activity Bar:
- **Theme Picker** - Visual grid of theme cards with live preview
- **Editor Settings** - Font family, size, line height, tab size
- **Terminal Settings** - Font family and size customization
- **UI Toggles** - Word wrap, minimap, font ligatures
- Real-time preview of all changes
- Settings persistence across sessions via localStorage

### File Management
Full file and folder management capabilities:
- **Context Menu** - Right-click on files/folders for actions:
  - New File (in folders)
  - New Folder (in folders)
  - Rename
  - Copy Path
  - Delete (with confirmation)
- **Header Toolbar** - Quick access buttons for:
  - New File
  - New Folder
  - Refresh
- **Inline Editing** - Rename files/folders directly in the tree
- Keyboard support: Enter to confirm, Escape to cancel

### Visual Identity
- **New SVG Logo** - Custom 3D logo with gradient colors and glow effects
- Unique visual identity distinct from VS Code

## Changed

### Visual Overhaul
Complete redesign of core UI components with distinctive Flux styling:

**Activity Bar**
- Gradient background with vertical accent line
- Diamond-shaped glowing indicator for active items
- Pulse glow animation on active items
- Scale-up effect on hover
- Settings icon rotates on hover

**Sidebar**
- Gradient background with decorative corner accent
- Glowing underline on headers
- Section headers styled as cards with borders
- Items have floating card style with hover slide effect
- Active items have glowing dot indicator
- Custom styled scrollbar with accent color on hover

**File Explorer**
- Gradient header with accent line
- Items have subtle borders and glow effects on selection
- Folder icons have warm glow effect
- Active files have glowing vertical line indicator
- File type-specific icon colors (JS yellow, TS blue, CSS purple, etc.)

**Status Bar**
- Subtle gradient background (replaced solid gradient fill)
- Glowing top border accent
- Animated accent dots on both edges
- Status items have borders and hover effects
- Branch item has special gradient styling
- Error/warning states have distinct colored gradients
- Flux branding badge with gradient text

**Editor Tabs**
- Rounded tab design with glow effects
- Active tab has glowing indicator

### Theme Integration
- Monaco Editor themes sync with selected theme
- Terminal (xterm.js) colors sync with selected theme
- All UI components respect theme CSS variables

## Technical

### New Files
- `/src/themes/` - Theme definitions for all 6 themes
- `/src/stores/theme-store.ts` - Zustand store with localStorage persistence
- `/src/components/ThemeProvider.tsx` - Applies theme CSS variables
- `/src/components/Settings/` - Complete settings modal UI
- `/src/components/ContextMenu/` - Reusable context menu component
- `/public/flux-icon.svg` - New SVG logo

### Dependencies
- Theme store uses Zustand with persist middleware
- Settings modal uses Lucide React icons
